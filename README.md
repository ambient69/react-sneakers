# ReactSneakers

### Technology stack

1. React + JavaScript
2. CSS/SASS
3. CSS Modules
4. Working with API - axios

### List of dependencies

1. react-router-dom
2. axios
3. react-content-loader
4. react-slick
5. sass

### Functionality

1. Adding an item to the cart
2. Adding a product to favorites
3. Placing an order + displaying orders on a separate page
